Создай UI дизайн для perenoska используя Stitch MCP.

Аргументы: $ARGUMENTS

Инструкции:
1. Используй скилл `stitch-design` для генерации дизайна
2. Учитывай контекст приложения:
   - Русскоязычный интерфейс
   - Цвета: WB — фиолетовый (#9B59B6), Ozon — синий (#005BFF), YM — жёлтый (#FFDB4D)
   - Основной поток: подключение аккаунтов → выбор товаров → превью → перенос → статус
3. После генерации покажи скриншот
4. Предложи как экспортировать результат в `app/static/`
