## Шаги

### Шаг 0: Автоматическая валидация

```bash
python .structure/.instructions/.scripts/validate-structure.py
```

Скрипт проверяет все правила S010-S012 и структурные проверки. Если валидация пройдена — **готово**, шаги 1-4.1 не нужны.

**Если скрипт недоступен** — выполнить шаги 1-4.1 вручную.

### Шаг 1: Проверить дерево

1. Открыть секцию "Дерево папок" в `/.structure/README.md`
2. Для каждой папки в дереве проверить:
   - Папка существует в файловой системе?
   - Комментарий соответствует краткому описанию в секции папки?
3. Для каждой папки в файловой системе проверить:
   - Папка есть в дереве?
   - Исключения: `node_modules/`, `.git/`, `dist/`

**Пример валидного дерева:**
```
├── .claude/                # Claude Code окружение
├── src/                    # Исходный код
└── specs/                  # Спецификации
```

### Шаг 2: Проверить синхронизацию

1. Сравнить **Оглавление** ↔ **Секции**:
   - Каждая папка из Оглавления имеет секцию в "Корневые папки"?
2. Сравнить **Секции** ↔ **Дерево**:
   - Каждая папка из секций есть в дереве?
3. Проверить **алфавитный порядок** во всех трёх местах

**Схема синхронизации:**
```
Оглавление ←→ Секции ←→ Дерево
     ↑_____________________↑
```

### Шаг 3: Проверить ссылки

1. Найти все ссылки `[текст](путь)` в документе
2. Для каждой ссылки проверить:
   - Файл по пути существует?
   - Если есть якорь `#section` — секция существует в целевом файле?
3. Убедиться, что формат пути относительный: `../папка/README.md`

**Пример валидной ссылки:**
```markdown
### 🔗 [.claude/](../.claude/README.md)
```

### Шаг 4: Проверить зеркала `.instructions`

> **ПРАВИЛО:** Папка в SSOT → зеркало в `.instructions`.

1. Для каждой корневой папки в SSOT проверить:
   - Существует `{папка}/.instructions/`?
   - Существует `{папка}/.instructions/README.md`?
2. Для каждой подпапки в SSOT проверить:
   - Существует `{корень}/.instructions/{подпуть}/`?
   - Существует `{корень}/.instructions/{подпуть}/README.md`?

**Пример:**
```
src/ в SSOT         → src/.instructions/README.md существует?
src/auth/ в SSOT    → src/.instructions/auth/README.md существует?
```

**Исключения:**
- Папки с префиксом `DELETE_` — помечены как удалённые
- Папка `.structure/` — не требует зеркала (сама содержит инструкции)

### Шаг 4.1: Проверить секцию "Вложенные области"

> **Если в папке `.instructions/` есть подпапки (кроме `.scripts/`):**

1. Секция "Вложенные области" существует?
2. Все подпапки перечислены в таблице?
3. Ссылки на README подпапок валидны?

**Типичные ошибки:**

| Код | Описание |
|-----|----------|
| S010 | Подпапка не указана в "Вложенные области" |
| S011 | Битая ссылка на README подпапки |
| S012 | Секция "Вложенные области" отсутствует при наличии подпапок |

## Чек-лист

- [ ] Дерево соответствует файловой системе
- [ ] Комментарии в дереве актуальны
- [ ] Оглавление ↔ Секции ↔ Дерево синхронизированы
- [ ] Алфавитный порядок соблюдён везде
- [ ] Все ссылки ведут на существующие файлы
- [ ] Все якоря ведут на существующие секции
- [ ] Все папки в SSOT имеют зеркало `.instructions`
- [ ] Секции "Вложенные области" соответствуют подпапкам

---

## Типичные ошибки

| Ошибка | Причина | Решение |
|--------|---------|---------|
| Битая ссылка | README удалён/перемещён | Создать README или убрать ссылку |
| Неверный якорь | Секция переименована | Исправить имя якоря |
| Папка не в дереве | Забыли добавить при создании | Добавить в дерево |
| Нет в Оглавлении | Забыли обновить | Добавить ссылку `- [папка/](#-папка)` |
| Нет секции | Забыли создать описание | Добавить секцию по стандарту |
| Рассинхрон порядка | Добавляли в разное время | Пересортировать все три места |
| Неверный комментарий | Описание папки изменилось | Синхронизировать с секцией |
| Нет зеркала `.instructions` | Забыли создать при добавлении папки | `mirror-instructions.py create {путь}` |
| Зеркало не обновлено | Папка переименована/перемещена | `mirror-instructions.py rename/move` |
| S010: Подпапка не указана | Забыли добавить в "Вложенные области" | Добавить строку в таблицу |
| S011: Битая ссылка | README подпапки не существует | Создать README подпапки |
| S012: Секция отсутствует | Есть подпапки, но нет секции | Добавить секцию "Вложенные области" |

## Скрипты

| Скрипт | Назначение | Инструкция |
|--------|------------|------------|
| [validate-structure.py](./.scripts/validate-structure.py) | Проверка дерева структуры | Этот документ |
| [validate.py](./.scripts/validate.py) | Единая валидация (структура + ссылки + .instructions) | Этот документ |
| [mirror-instructions.py](./.scripts/mirror-instructions.py) | Создание/исправление зеркал `.instructions` | [create-structure.md](./create-structure.md) |

**Использование:**
```bash
# Проверка структуры
python .structure/.instructions/.scripts/validate-structure.py
python .structure/.instructions/.scripts/validate-structure.py --check-instructions

# Единая валидация
python .structure/.instructions/.scripts/validate.py
python .structure/.instructions/.scripts/validate.py --path <папка>

# Создание недостающего зеркала
python .structure/.instructions/.scripts/mirror-instructions.py create <путь>
```
