## Шаги

### Шаг 0: Автоматическая валидация

```bash
python .instructions/.scripts/validate-instruction.py {path}
```

Скрипт проверяет все правила I001-I043. Если валидация пройдена — **готово**, шаги 1-5 не нужны.

**Если скрипт недоступен** — выполнить шаги 1-5 вручную.

### Шаг 1: Проверить файл

| Проверка | Обязательно | Описание |
|----------|:-----------:|----------|
| Расширение | ✅ | `.md` |
| Имя файла | ✅ | kebab-case, латиница |
| Расположение | ✅ | В папке `.instructions/` |
| Кодировка | ✅ | UTF-8 |

### Шаг 2: Проверить frontmatter

| Проверка | Обязательно | Описание |
|----------|:-----------:|----------|
| Наличие | ✅ | Блок `---` в начале |
| `description` | ✅ | Присутствует, не пустое |
| `standard` | ✅ | Ссылка на стандарт |
| `index` | ✅ | Ссылка на README области |

**SSOT:** [standard-frontmatter.md](/.structure/.instructions/standard-frontmatter.md)

### Шаг 3: Проверить структуру

| Проверка | Обязательно | Код | Описание |
|----------|:-----------:|:---:|----------|
| Заголовок H1 | ✅ | I020 | Единственный `#` |
| Описание после H1 | ✅ | I025 | Текст после заголовка |
| Полезные ссылки | ✅ | I024 | Блок `**Полезные ссылки:**` |
| Связанные документы | ✅ | I023 | Таблица с 4 типами |
| Оглавление | ✅ | I022 | `## Оглавление` |
| Разделы непустые | ✅ | — | Каждый раздел имеет хотя бы одну секцию |
| Нумерация секций | ⚠️ | I026 | Только для standard-инструкций |
| Нумерация шагов | ⚠️ | I027 | Только для create/modify/validation |

### Шаг 4: Проверить ссылки

| Проверка | Обязательно | Описание |
|----------|:-----------:|----------|
| Валидность | ✅ | Файлы существуют |
| Относительные | ⚠️ | Предпочтительны |
| Якоря | ⚠️ | Секции существуют |

### Шаг 5: Проверить правила create/modify

> **Только для инструкций типа create, modify, validation.**

| Проверка | Обязательно | Код | Описание |
|----------|:-----------:|:---:|----------|
| Порядок создания | ✅ | I043 | Предыдущие типы в цепочке существуют |
| Проверка существующих | ✅ | I040 | Есть шаг "Проверить существующие" (для create) |
| Отчёт | ✅ | I041 | Есть шаг "Отчёт" (для create/modify) |
| Секция "Скиллы" | ✅ | I042 | Непустая (для create/modify/validation) |

**Порядок создания (I043):**

| Тип инструкции | Требуется |
|----------------|-----------|
| `validation-{object}.md` | `standard-{object}.md` |
| `create-{object}.md` | `standard-{object}.md` + `validation-{object}.md` |
| `modify-{object}.md` | все предыдущие типы |

## Чек-лист

### Файл
- [ ] Расширение `.md`
- [ ] Имя в kebab-case с правильным префиксом
- [ ] Расположение в `.instructions/`

### Frontmatter
- [ ] Присутствует блок `---`
- [ ] Поле `description` заполнено
- [ ] Поле `standard` указывает на стандарт
- [ ] Поле `index` указывает на README

### Структура
- [ ] Один заголовок H1 (I020, I021)
- [ ] Описание после заголовка (I025)
- [ ] Блок "Полезные ссылки" (I024)
- [ ] Таблица "Связанные документы" (I023)
- [ ] Секция "Оглавление" (I022)
- [ ] Все разделы непустые
- [ ] Секции пронумерованы для standard-инструкций (I026)
- [ ] Шаги пронумерованы для create/modify/validation (I027)

### Ссылки
- [ ] Все ссылки валидны
- [ ] Все якоря существуют

### Правила create/modify
- [ ] Порядок создания соблюдён (предыдущие типы существуют)
- [ ] Есть шаг "Проверить существующие" (для create)
- [ ] Есть шаг "Отчёт" (для create/modify)
- [ ] Секция "Скиллы" непустая (для create/modify/validation)

## Типичные ошибки

| Ошибка | Код | Причина | Решение |
|--------|-----|---------|---------|
| Неверное расширение | I001 | Не `.md` | Переименовать файл |
| Неверное имя | I002 | Не kebab-case | Переименовать файл |
| Неверное расположение | I003 | Не в `.instructions/` | Переместить файл |
| Нет frontmatter | I010 | Отсутствует блок `---` | Добавить frontmatter |
| Нет description | I011 | Пустое поле | Заполнить описание |
| Нет standard | I012 | Не указан стандарт | Добавить ссылку на стандарт |
| Нет index | I013 | Не указан README | Добавить ссылку на README |
| Нет заголовка H1 | I020 | Отсутствует `#` | Добавить заголовок |
| Несколько H1 | I021 | Больше одного `#` | Оставить один |
| Нет оглавления | I022 | Нет `## Оглавление` | Добавить оглавление |
| Нет связанных документов | I023 | Нет таблицы | Добавить таблицу "Связанные документы" |
| Нет полезных ссылок | I024 | Нет блока | Добавить `**Полезные ссылки:**` |
| Нет описания | I025 | Нет текста после H1 | Добавить описание |
| Ненумерованные секции | I026 | standard-инструкция без номеров | Добавить номера секций: `## 1. Секция` |
| Нет шагов | I027 | create/modify/validation без шагов | Добавить `### Шаг N: {Действие}` |
| Битая ссылка | I030 | Файл не существует | Исправить путь |
| Битый якорь | I031 | Секция не существует | Исправить якорь |
| Нет проверки существующих | I040 | create без шага проверки | Добавить шаг "Проверить существующие" |
| Нет отчёта | I041 | create/modify без отчёта | Добавить шаг "Отчёт" |
| Пустая секция "Скиллы" | I042 | Нет связанного скилла | Создать скилл через `/skill-create` |
| Нарушен порядок создания | I043 | Предыдущий тип не существует | Сначала создать предыдущий тип в цепочке |

## Скрипты

| Скрипт | Назначение | Инструкция |
|--------|------------|------------|
| [validate-instruction.py](./.scripts/validate-instruction.py) | Валидация формата инструкции | Этот документ |

**Использование:**
```bash
# Проверка одного файла
python .instructions/.scripts/validate-instruction.py path/to/instruction.md

# Проверка всех инструкций
python .instructions/.scripts/validate-instruction.py --all

# JSON вывод
python .instructions/.scripts/validate-instruction.py --json path/to/instruction.md
```

**Ожидаемый вывод:**
```
✅ instruction.md — валидация пройдена
❌ broken.md — 3 ошибки:
   I010: Отсутствует frontmatter
   I020: Отсутствует заголовок H1
   I030: Битая ссылка: ./nonexistent.md
```
