# Backlog Notes

## Технические ограничения MVP

- нет фонового воркера для импорта;
- нет retry и rate limiting;
- нет истории ручного маппинга;
- нет полноценной категорийной матрицы;
- нет интеграционных sandbox-тестов WB/Ozon;
- SQLite подходит для локальной разработки, но не для серьёзной нагрузки.

## Логичные следующие шаги

- сохранить таблицу соответствий категорий и ручных атрибутных правок;
- добавить хранение и проксирование медиа;
- усилить обработку ошибок marketplace API;
- покрыть реальными интеграционными сценариями.
