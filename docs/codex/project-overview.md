# Project Overview

## Назначение

`perenoska` — локальный FastAPI-сервис, который помогает переносить товарные карточки между Wildberries и Ozon.

## Что уже реализовано

- регистрация и логин пользователей;
- сохранение подключений WB/Ozon;
- шифрование чувствительных данных;
- загрузка списка товаров и категорий;
- preview будущего импорта;
- запуск переноса и синхронизация статуса задач;
- встроенный минималистичный UI.

## Текущий формат проекта

- один Python-пакет `app`;
- SQLite без ORM и без миграционного фреймворка;
- backend и UI лежат в одном репозитории;
- тестирование построено на `pytest` и `fastapi.testclient`.

## Что важно понимать Codex

- Это MVP, поэтому часть решений намеренно простая.
- Здесь много логики, связанной с преобразованием данных маркетплейсов.
- Для большинства изменений ключевые точки входа: `app/main.py`, `app/api/routes/`, `app/services/`, `app/static/app.js`.
